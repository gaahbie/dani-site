# Project Summary
O projeto "Alfa Soluções em Concreto" é um site de catálogo digital que apresenta produtos de concreto pré-moldado, como móveis e revestimentos, voltado para a construção civil. A plataforma proporciona uma experiência visual e organizada para os clientes, permitindo a exploração de uma ampla gama de produtos através de um design responsivo. Recentemente, o site foi atualizado para incluir um favicon personalizado com o logo da empresa, melhorando a identidade visual.

# Project Module Description
O projeto é dividido em várias funcionalidades principais:
- **Homepage**: Apresenta um banner principal, seções de categorias e uma grade de produtos.
- **Página de Detalhes do Produto**: Fornece informações detalhadas sobre cada produto, incluindo descrições, dimensões e imagens.
- **Busca e Filtros**: Permite que os usuários filtrem produtos por categoria e busquem por palavras-chave.
- **Integração com WhatsApp**: Facilita a comunicação com os clientes para pedidos e orçamentos.

# File Description Inventory
- **README.md**: Documentação do projeto.
- **package.json**: Gerenciador de dependências.
- **src/App.tsx**: Componente principal do aplicativo.
- **src/pages/**: Contém as páginas do aplicativo, incluindo a homepage e a página de detalhes do produto.
- **src/components/**: Componentes reutilizáveis como cabeçalho, rodapé e cartões de produtos.
- **src/data/products.ts**: Contém dados dos produtos e categorias.

# Usage
Para iniciar o projeto:
1. Instale as dependências:
   ```bash
   pnpm install
   ```
2. Execute o lint para verificar o código:
   ```bash
   pnpm run lint
   ```
3. Construa o projeto:
   ```bash
   pnpm run build
   ```
